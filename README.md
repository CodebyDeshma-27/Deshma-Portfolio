# Deshma Portfolio

A personal portfolio website showcasing my background, skills, certifications, internship experience, and projects as I continue my journey in Cybersecurity, DevOps, and Full Stack Development.

## Tech Stack

- React
- TypeScript
- Vite
- Tailwind CSS

## Features

- Modern responsive UI
- About Me
- Skills
- Experience
- Certifications
- Resume Preview & Download
- Coding & Security Profiles
- Contact Information

## Getting Started

Clone the repository:

```bash
git clone https://github.com/CodebyDeshma-27/Deshma-Portfolio.git
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open:

```
http://localhost:8080
```
